## A 3D render test.

Implemented:
- Wavefront .obj and .mtl file loading.
- Textured mesh rendering.
- Basic Phong lighting.
- Armature loading from a custom file format (.bau).
- - Blender export plugin for this format.
- Model skinning.
- Posable mesh.
